Current Issues with SFGOS

While we work hard to provide the best experience, some issues have been reported by users. Below is a list of known issues along with workarounds where available.

1. Audio Stuttering in Some Games

Some users have reported audio stuttering in specific games. This issue seems to be related to the sound driver.

Workaround: Try updating your sound driver or switching to a different audio output device.

2. Graphics Driver Incompatibility with Latest NVIDIA GPUs

Newer NVIDIA graphics cards may not perform optimally on the initial SFGOS install.

Workaround: Install the latest proprietary NVIDIA drivers manually. Refer to the Arch Wiki for more details.

3. System Freezing After Resume from Sleep

There have been cases where systems freeze after resuming from sleep mode.

Workaround: This issue may be resolved by disabling hybrid sleep or adjusting kernel parameters. Please check the Tips & Tricks Wiki.

4. Performance Drops in Multiplayer Games

Some multiplayer games show performance drops when playing online with high ping.

Workaround: Set the gameï¿½s network settings to reduce packet loss, or try using a VPN for more stable connection.
