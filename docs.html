Installation Guide

To install SFGOS, follow these steps:

  1. Download the SFGOS ISO from the official SFGOS website.
  2. Create a bootable USB drive.
  3. Boot from the USB drive and follow the on-screen instructions to install the operating system.
  4. Once installed, configure your system as per your preferences.

Configuration Guide

Once SFGOS is installed, you can configure your system as follows:

  • Set up networking and update your mirror list.
  • Install drivers and additional software packages as needed.
  • Customize your desktop environment, settings, and preferences.

Troubleshooting

If you encounter issues, refer to the following resources:

  • Check the Known Issues page.
  • Visit the SFGOS Discord server for support.
